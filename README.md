営業プロセスボード Demo

概要
SES営業における営業プロセス全体を可視化し、ボトルネックの特定と改善アクションにつなげるためのプロセス管理ツール

背景
・Excel管理では営業プロセス全体の把握が難しい
・返信率 / 面談率 / 成約率などのプロセス指標が可視化されていない
・営業活動が属人化し、改善ポイントが見えにくい

目的
営業プロセスを分解し、どこに課題があるのかを明確にし、改善につなげる

機能
・KPIサマリー表示（件数ベース）
・売上 / 粗利の可視化
・営業プロセスのファネル管理
・ボトルネックの可視化
・月別切替機能
・診断指標（各プロセスの率分析）

技術
・HTML / CSS / JavaScript
・GitHub Pagesで公開

工夫した点
・営業プロセス単位で分解し、改善できる設計
・一目でボトルネックが分かるUI設計
・実務で使うことを前提としたKPI設計

データ連携
・Googleスプレッドシートと連携（本番環境）
・デモ環境ではサンプルデータを使用

実運用想定
・営業活動の入力 → スプレッドシート
・営業プロセスボードでリアルタイム可視化
・各プロセスの率・粗利・進捗を即時確認

特徴
・単なる可視化ではなく「改善までできる」設計
・営業のボトルネックを特定できる
・データ更新のみで自動反映

デモについて
本リポジトリはデモ版（サンプルデータ）です。
実運用ではスプレッドシート連携によりリアルタイムで反映されます。

・個人別KPI / 粗利目標管理（本番環境）
・ログイン機能追加

# 営業KPIダッシュボード

## 概要
SES営業におけるKPIを可視化するためのダッシュボード

## 背景
・Excel管理では全体把握が難しい  
・返信率、成約率などの可視化ができていなかった

## 目的
営業活動のボトルネックを特定し、改善につなげる

## 機能
・KPIサマリー表示
・売上 / 粗利表示
・月別切替
・各指標の可視化

## 技術
・HTML / CSS / JavaScript
・GitHub Pagesで公開

## 工夫した点
・営業視点で必要な指標を整理
・一目で状況が分かるUI設計
・実務で使う前提で設計

## 今後の改善
・スプレッドシート連携
・リアルタイム更新
・ログイン機能追加
